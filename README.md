# AI Course Projects

This repository showcases a series of projects developed as part of an Artificial Intelligence course.

The course emphasizes both theoretical foundations and practical implementations of AI techniques. The assignments demonstrate the ability to model complex problems, design efficient algorithms, and solve them using methods from search, probabilistic reasoning, and logic.

### Key areas:
- Search algorithms and heuristic optimization
- Stochastic decision-making and Markov Decision Processes (MDPs)
- Logical reasoning and SAT solving
- Constraint-based problem solving

The projects reflect strong analytical thinking, algorithmic design skills, and the ability to handle both deterministic and uncertain environments.
